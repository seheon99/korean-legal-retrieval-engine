## Code landed

- [scripts/fetch_law_samples.sh](../../scripts/fetch_law_samples.sh) —
  first code in the project. Three modes: zero-arg (Phase-1 default set),
  `--doc <law_id> <mst>`, `--search <query>`. Idempotent (skip on
  existing unless `--force`), atomic (tmp + rename), single dependency
  (`curl`). Document fetches → `data/raw/{law_id}/{mst}.xml`; search
  fetches → `docs/api-samples/`. Live-tested end-to-end on Fedora.
