## Notable empirical findings

- The 중대재해처벌법 has **no 시행규칙** — the API returns only Act + 시행령
  for the keyword "중대재해". Phase 1 statute scope is therefore Act + 시행령
  + the 시행령's appendices. Captured in TODO-3.
- 중대재해처벌법 시행령 has **5 annexes, 0 forms**. Forms-table behavior
  was validated against 산안법 시행규칙 as a cross-reference (27 annexes,
  111 forms; not retained because outside Phase 1 scope).
- The XML field `<별표편집여부>` is misleading — set to `N` on a 시행령 that
  has 5 annexes. Don't trust it as a "has annexes?" signal.
- The XML uses `<조문여부>` to distinguish chapter headings (`전문`) from
  actual articles (`조문`). Same `조문번호` can appear in both — `조문키`
  disambiguates (`0003000` heading vs `0003001` article).
