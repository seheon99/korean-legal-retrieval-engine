## Empirical findings worth preserving

- 중대재해처벌법 has no 시행규칙. The API returns Act + 시행령 only for the
  keyword "중대재해". Captured in TODO-3.
- 중대재해처벌법 시행령 has 5 annexes, 0 forms.
- `<별표편집여부>` is misleading — set to `N` on a 시행령 with 5 annexes.
  Do not trust it as a "has annexes?" signal.
- `<조문여부>` distinguishes chapter headings (`전문`) from articles (`조문`).
  Same `조문번호` can appear in both; `조문키` disambiguates
  (`0003000` heading vs `0003001` article).
- Forms have inline body text but it is ASCII box-art unsuitable for
  retrieval; split confirmed on retrieval-quality grounds, not data absence.
